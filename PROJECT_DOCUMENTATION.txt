================================================================================
                              PROJECT OVERVIEW
================================================================================

PathFinder Pro is an AI-powered career guidance platform that helps users 
discover their ideal career path based on their skills, interests, personality 
type (Holland Code), and mental health assessment scores.

Key Features:
- Career Discovery Assessment (5-step onboarding)
- AI-powered Career Advice using Google Gemini
- Real-time Job Search with JSearch API
- Course Recommendations (Coursera, Udemy, LinkedIn Learning, edX)
- GAD-7 and PHQ-9 Mental Health Assessments
- Personalized Career Matching based on Holland Code (RIASEC)
- AI Chat Assistant for career guidance
- User Authentication with Supabase

                           HOW TO RUN

FRONTEND:
---------

# Runs on http://localhost:5173

BACKEND:
--------
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
# Runs on http://localhost:8000

================================================================================
                           FEATURES BREAKDOWN
================================================================================

1. CAREER DISCOVERY (5-Step Onboarding)
   - Step 1: Basic Info (name, education, target role, skills)
   - Step 2: Interests (Holland Code - RIASEC personality)
   - Step 3: GAD-7 Anxiety Assessment (AI-personalized)
   - Step 4: PHQ-9 Mood Assessment (AI-personalized)
   - Step 5: AI Career Advice with:
     * Career Matches
     * Course Recommendations
     * Job Search Links
     * Work Environment Tips
     * Action Steps

2. AI ASSISTANT
   - Real-time chat with career coach
   - Job search by keywords/location
   - Course recommendations
   - Career guidance

3. AUTHENTICATION
   - Email/Password signup
   - Login
   - Password reset (PKCE flow)
   - Session management

4. JOB SEARCH
   - Real-time job listings from JSearch API
   - Filter by role, location
   - Direct apply links

5. COURSE RECOMMENDATIONS
   - Coursera links
   - Udemy links
   - LinkedIn Learning links
   - edX links
